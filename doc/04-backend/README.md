# 后端开发文档

本目录存放后端开发相关文档，由后端开发工程师维护。

## 子目录说明

| 目录 | 说明 |
|------|------|
| `api/` | 接口定义文档 (OpenAPI/Swagger) |
| `database/` | 数据库设计文档 |
| `services/` | 微服务/模块设计说明 |
| `deployment/` | 部署与运维文档 |

## 核心模块

### PDF 转换服务
- PDF ↔ Word 转换
- PDF ↔ Excel 转换
- PDF ↔ PPT 转换
- PDF ↔ 图片转换

### PDF 编辑服务
- PDF 合并
- PDF 拆分
- PDF 压缩
- PDF 页面管理
